Smart Medical Report Analyzing Platform.

## Project Overview
A simple Website/platform where patients and doctors can login separately.
Patients can provide their pre-medical history, current report and specify allergies (if any), so that the patient's medical history is saved,
they can chat with our chatbot (With respect to the Patient's data our basic AI chatbot responds and provides basic general suggestions).
The patient can connect with the available doctors(live).

This is a beginner-friendly full-stack + basic AI project, built using HTML, CSS, JavaScript, Django, MySQL, and Gemini API.

## Aims of the Project
1) Help patients store and manage medical history
2) Provide a chatbot that gives safe, general guidance
3) Suggest consulting doctors registered on the platform
4) Allow live chat between patient and doctor
5) Add more AI/ML features in future version
